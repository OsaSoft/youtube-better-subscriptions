What's New in v0.23.0

LATEST
  • Option to hide livestreams (separate from premieres)
  • Option to hide members-only videos
  • Option to hide the "Mark as watched" button
  • SHA256 checksums added to releases for integrity verification

Want to Help?

This extension is developed completely as a hobby project in my free time. If you want to help me, even reaching out with a kind word is enough.

I would appreciate it if you can help spread the word and get the extension out into more people's hands.

For those that would like a more hands-on approach, you can contribute to the source code on GitHub. If you can't code, there are some non-coding things I'd very much appreciate help with (like prettier UI, screenshots, etc). Check out the issues with the "Help Wanted" tag.
