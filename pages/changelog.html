Thanks for using Better Subscriptions for YouTube™!

I hope you like this extension. Feel free to reach out to me on Twitter @Osa_Soft or if you find any bugs or have a cool idea for a new feature, let me know on the extension's GitHub page!

The purpose of this page is to let you know what's changed or new in the extension and it will show up after installing or updating the extension to a new version.

What's new in version 0.19.0?

  • Sync watch history across devices (using same browser)

What's new in version 0.18.0? (repeating this here, as 0.18.0 did not get published for Chrome)

  • Enable UI for channel's live video (livestream) history
  • Moved forced-grid rules in css to improve performance
  • Enabled forced grid layout on all page
  • Fix: shorts video query catch "Latest YouTube posts" too

Contributor thank you

Massive kudos to Valentin Mocanu for the tackling the sync storage, which has been on my list to do since the very first version of this extension.

Want to help?

This extension is developed completely as a hobby project in my free time, if you want to help me, even reaching out with a kind word is enough.

Other than that, I would appreciate it if you can help spread the word and get the extension out into more people's hands.

For those that would like a more hands-on approach to helping, you can always contribute to the source code on GitHub, and if you can't code, there are some non-coding things I'd very much appreciate some help with (like prettier UI, screenshots, etc). You can see some of those in the Github issues with the "Help Wanted" tag
